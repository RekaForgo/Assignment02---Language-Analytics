# Assignment02, Language-Analytics
Repository for code related to assignment 02 for Language analytics course, Spring 2026, Cultural Data Science Supplementary Subject

## How to run
In the terminal, navigate to the assignment folder that you downloaded.
then run: **pip install -r requirements.txt**
And then: **python src/main.py --test_data data/TOP_SECRET_fake_real_news_test_data.csv**. 
(*this might take some time, depending on your machine.*)

Output will be saved to the folder called "out" and your files will be timestamped with the current data and time.
